7. When should accessor methods be used?
Returning objects through interfaces: When a method returns an object that implements a specific interface.
Accessor methods are used in Java to provide controlled access to the private fields of a class. They allow 
other classes to retrieve the values of these fields without directly accessing them, thereby promoting 
encapsulation and data hiding.
